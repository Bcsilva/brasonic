The player fell off from the level, and then?<\/strong>
Since this version had not programmed a half of the character to die by falling out of it, you can press the “R” key to return from the last checkpoint position.

BrazSonic 2 2013 BETA<\/h2>\n\n\n\n

How to deal with bugs and bugs that happen throughout the game?<\/h3>\n\n\n\n

Below I’ve listed some bugs and bugs that might happen during this release. Because it is not a distributable version, do not be alarmed if any of them occur to you!<\/p>\n\n\n\n

Here we go…<\/p>\n\n\n\n

“Trying to use non-existing surface”:<\/strong>
On the dialog error box, click “Ignore”, or hold “I” key, until the error stop to keep showing. It uses to happen on the start screen and “new game” start.

“Background does not exists”: <\/strong>
Close the game and try running again.

Level music does not play: <\/strong>
This is a bug that still persists on some computers, happened even with my own PC, I’m still looking for the solution to this. When the stage music does not play, and you pick up some “power-up” of invincibility or speed, you can play, in infinite loop, their music or the sound of the jump.

Backgrounds are not displaying:<\/strong>
For some reason, that I have not discovered yet, if the “Background does not exists” error does not appear the game may at some point lose the link with the external files of the phase sceneries and thus not load the sceneries parts from the front of the level (ground, loopings, ramps, etc.). This may stop appearing as soon as you change level.

Foregrounds are not displaying:<\/strong>
For the same reason above, perhaps due to some software bug where the game was developed, the scenario of the front (floor, ramps, etc.), if it does not present an error message, it can load with its disfigured pieces with strange colors. This does not affect the course of the phase, only the aesthetics.

General crashing:<\/strong>
Tends to happen after a time of play, the game freezes and the scenery may be all disfigured. This tends to happen more often with PeloTails and Baianuckles. In that case, I recommend closing the game and trying to reopen the game. If it persists, I recommend trying on another machine.

The Brazil map is not being displayed on the level select screen, just the water:<\/strong>
Try pressing “S” to go back and choose the same character to proceed to the map to appear again. This is a common bug in this version, which does not affect the game itself, just the aesthetics.<\/p>\n\n\n\n
