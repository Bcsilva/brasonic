I found these days, after digging into my old backups, 2 old builds of BrazSonic 2 from 2005, and 2006. Both versions had been pending for a long time on the site btw.<\/p>\n\n\n\n

The first, from 2005, is a prototype of BrazSonic 2 Special Stage, which uses a third-person 3d engine, but using fake 3d feature, because of the Game Maker 5 limitations of that time. Visually, it looked like the old DOOM games a bit. The prototype has 2 and a half levels (the third is quite unfinished to be called “level” rs). In that, BrazSonic runs through a maze dodging obstacles such as cars and trees and collecting coins in search of the Mensal\u00e3o briefcase, a common Brazilian political scheme of that time.<\/p>\n\n\n\n
